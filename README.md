# DriverCity
Водительское приложения сервиса ВсёТакси

## Описание
Водительское приложение для регистрации в сети сервиса ВсёТакси с возможностью выхода на линию и принятия заказа по заданнному радиусу.

## Использованные библиотеки
* Retrofit
* Coroutines
* Dagger-Hilt
* Architecture Components (LiveData, LifeCycle and ViewModel)
* LocationServices (Google services)
* Work Manager (Dagger-hilt)
